How it Works

QuickSort is a Divide-and-Conquer algorithm. It partitions the array into two sub-arrays by selecting a pivot element. Elements smaller than the pivot go to the left, and elements greater go to the right. The sub-arrays are then sorted recursively.

Key Optimizations Used Here

  • Median-of-Three: Selects the median of the first, middle, and last elements to avoid the \(O(N^2)\) worst case on sorted arrays.
  • Insertion Sort Threshold: Small sub-arrays (size < 10) fallback to insertion sort to avoid recursive overhead.
